## API With Pydantic

Application fastapi with pydantic 

### Requirement
- Python ^3.9

### Installation 
- clone project
- create virtual env with following command (ubuntu or popos)
  ```
  python3 -m venv venv
  ```
- activate the virtual env with :
  ```
  . venv/bin/activate
  ```  
- pip install -U pip & pip install -r requirements.txt

### Running

#### API service  
```
python3 main.py
```
